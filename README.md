# Eurobeat CLI APP
## Descripción
Esta aplicación consiste en un servidor que sirve letras de canciones del género "Eurobeat" a un cliente que las solicita mediante linea de comandos. Usa la arquitectura **cliente-servidor** y la motivación principal es hacer un homenaje a la música de los 80's/90's. 
Las habilidades principales que se pusieron a prueba son Linux, Python y *Command Line*, junto con conocimientos de programación básicos.
